Curso - Lógica de programação com Javascript

Variáveis são espaços alocados na memoria para armazenar informação; Podendo ser numero real ou inteiro, string (texto), booloean (verdadeiro ou falso), NaN / Not a Number (nao número), null (nulo), undefined, array, ou objeto.
Devem obrigatoriamente começar com uma letra minúscula e os demais caracteres podem ser de quase qualquer tipo

  • O comando prompt() apresenta um caixa de dialogo e captura uma informaçao do usuário.
  • alert() apresenta um pop up com uma informação qualquer.
